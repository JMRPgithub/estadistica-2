	 sol:\\

	u\\

	n = 36\\
	P (u-0.1 \le x \le u+0.1) = P( x \le u+0.1)- P( x \le u-0.1)\\
	=P( x-u/o/sqrt(n) \le u+0.1-u/)- P( x-u/o/sqrt(n) \le u-0.1-u)\\
	=P( z \le 2)- P( z \le -2)\\
	=0.97725-0.02275
